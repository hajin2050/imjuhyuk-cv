# 임주혁 · Im Juhyuk — Web CV

사업 기획자 · 스타트업 빌더 · 리더 인터랙티브 웹 이력서.

- 묵직한 다크 차콜 톤 · 두꺼운(900) 웨이트 · 큰 타이포/큰 사진 · 커맨드 레드 포인트
- 큰 숫자 카운트업, 경력 타임라인, 스크롤 리빌, 키워드 마키
- 의존성 없는 단일 `index.html`

## 로컬 미리보기
```bash
python3 -m http.server 4181   # http://localhost:4181
```
